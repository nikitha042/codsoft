Titanic Survival Prediction

Overview

This project uses Machine Learning to predict whether a passenger survived the Titanic disaster based on passenger information such as age, gender, fare, and passenger class.

Objective

To build a predictive model that can determine passenger survival using the Titanic dataset.

Tools and Technologies

- Python
- Pandas
- Scikit-learn
- VS Code
- GitHub

Dataset

The Titanic dataset contains passenger details including:

- Passenger Class (Pclass)
- Gender (Sex)
- Age
- Number of Siblings/Spouses (SibSp)
- Number of Parents/Children (Parch)
- Fare
- Survival Status (Survived)

Machine Learning Model

Random Forest Classifier

Steps Performed

1. Loaded the dataset using Pandas.
2. Handled missing values.
3. Converted categorical data into numerical format.
4. Selected important features.
5. Split the dataset into training and testing sets.
6. Trained a Random Forest Classifier.
7. Evaluated model performance.

Results

- Model Accuracy: 80.44%

Conclusion

The Random Forest Classifier successfully predicted passenger survival with an accuracy of 80.44%. This project demonstrates the application of data preprocessing, feature selection, and machine learning techniques for predictive analytics.
