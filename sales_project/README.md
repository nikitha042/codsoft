Sales Prediction Using Python

Project Overview

This project uses Machine Learning to predict product sales based on advertising expenditures on TV, Radio, and Newspaper platforms.

Objective

To build a predictive model that estimates sales using advertising budget data.

Technologies Used

- Python
- Pandas
- Matplotlib
- Scikit-learn

Dataset

The dataset contains the following columns:

- TV
- Radio
- Newspaper
- Sales

Machine Learning Algorithm

- Linear Regression

Project Workflow

1. Import required libraries
2. Load the dataset
3. Explore and visualize data
4. Split data into training and testing sets
5. Train the Linear Regression model
6. Predict sales values
7. Evaluate model performance using R² Score

Results

The model predicts sales based on advertising spending and achieves a high R² score, indicating good prediction performance.

How to Run

1. Install required libraries:
   pip install pandas matplotlib scikit-learn

2. Place the dataset file in the project folder.

3. Run:
   python sales_prediction.py
