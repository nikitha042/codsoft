Movie Rating Prediction with Python

Project Overview

This project focuses on predicting movie ratings using Machine Learning techniques. The model analyzes movie-related features and predicts ratings based on available data.

Objective

To build a predictive model that estimates movie ratings using historical movie information and audience voting data.

Technologies Used

- Python
- Pandas
- Matplotlib
- Scikit-learn

Dataset Features

The dataset contains:

- Name
- Year
- Duration
- Genre
- Rating
- Votes
- Director
- Actor 1
- Actor 2
- Actor 3

Machine Learning Algorithm

- Linear Regression

Project Workflow

1. Data Collection
2. Data Cleaning and Preprocessing
3. Exploratory Data Analysis
4. Feature Selection
5. Model Training
6. Rating Prediction
7. Model Evaluation

Results

The model was trained on movie data and evaluated using performance metrics such as:

- Mean Squared Error (MSE)
- R² Score

The model successfully predicts movie ratings based on the available features.

How to Run

Install Required Libraries

pip install pandas matplotlib scikit-learn

Run the Project

python movie_rating_prediction.py
