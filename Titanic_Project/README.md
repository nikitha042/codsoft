Titanic Survival Prediction

Project Overview

This project uses Machine Learning to predict whether a passenger survived the Titanic disaster based on passenger information such as age, gender, ticket class, fare, and family details.

Objective

To build a predictive model that determines the survival probability of Titanic passengers using historical data.

Technologies Used

- Python
- Pandas
- Matplotlib
- Scikit-learn

Dataset Features

The dataset includes:

- PassengerId
- Survived
- Pclass
- Name
- Sex
- Age
- SibSp
- Parch
- Ticket
- Fare
- Cabin
- Embarked

Machine Learning Algorithm

- Logistic Regression

Project Workflow

1. Data Collection
2. Data Cleaning and Preprocessing
3. Handling Missing Values
4. Feature Selection
5. Train-Test Split
6. Model Training
7. Survival Prediction
8. Model Evaluation

Results

The Logistic Regression model was trained on the Titanic dataset and used to predict passenger survival. The model performance was evaluated using accuracy metrics.

How to Run

Install Required Libraries

pip install pandas matplotlib scikit-learn

Run the Project

python titanic_survival_prediction.py
